Linux下安装nginx

Posted on 2019-12-06

#学习笔记

今天学习了在Linux系统中nginx的安装。安装步骤如下:（ps:因为还不会使用markdown插图，所以今天没有截图…）
1、到官网下载nginx;（http://nginx.org/en/download.html）
2、安装工具包的下载
yum install -y wget
yum install -y vim-enhanced
yum install -y make cmake gcc gcc-c++
3、下载nginx所需要的一些依赖
yum install -y pcre pcre-devel
yum install -y zlib zlib-devel
yum install -y openssl openssl-devel
4、解压nginx-1.14.2.tar.gz到/usr/local/目录下
tar -zxvf nginx-1.14.2.tar.gz
5、进入nginx-1.14.2执行./configure配置
[root@localhost nginx-1.14.2]# ./configure
6、编译安装
[root@localhost nginx-1.14.2]# make && make install
7、安装成功，因为nginx需要使用80端口访问，Linux端口默认都是关闭的，所以要先打开80端口才能访问；
[root@localhost nginx-1.14.2]# make && make install–查询所有开放端口
sudo firewall-cmd --add-port=80/tcp --permanent–修改80端口为开放端口
firewall-cmd --reload–重启防火墙
